## 待开发功能

### 管理员管理后台

**优先级**: 中

**描述**: 开发管理员管理后台，用于管理远程仓库笔记的删除操作。

**背景**:
- 远程仓库（GitHub等）同步的笔记在前端不显示删除按钮
- 普通用户无法删除远程仓库的笔记
- 删除功能保留在后端，需要管理员权限才能执行

**功能需求**:

1. **管理员认证**
   - 管理员登录系统
   - 权限验证机制

2. **笔记管理**
   - 查看所有笔记列表（包括远程仓库笔记）
   - 删除远程仓库笔记
   - 查看笔记详情

3. **仓库管理**
   - 查看已连接的远程仓库列表
   - 管理仓库同步状态
   - 删除仓库及其关联笔记

**技术实现**:
- 后端 API 已保留删除功能
- 需要添加管理员角色和权限系统
- 需要开发管理后台前端界面

**相关代码位置**:
- `client/src/views/HomeView.vue` - `show-delete-button` 条件判断
- `client/src/components/common/FileTree.vue` - 文件树删除按钮条件
- `server/src/notes/notes.controller.ts` - 笔记删除 API

**状态**: 待开发
